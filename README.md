# Trybe Project - Docker To do List

## 📝 About

The project **🐋 Docker To do List** is a "containerization" of frontend, backend and test applications was implemented to practice Docker, developed in the Trybe course.

## 🚀 Built With

[![React][React]][React-url][![Javascript][Javascript]][Javascript-url][![HTML5][HTML5]][HTML5-url][![CSS3][CSS3]][CSS3-url][![Jest][Jest]][Jest-url]

## 📌 Skills developed

In this project, I was able to:

- Use dockers commands in the CLI - Command line interface;
- Create a Docker container for a front-end application;
- Create a Docker container for a back-end application;
- Create a Docker container for a testing application;
- Orchestrate the three containers using Docker compose.
  
## ⬇️ Installing dependencies


  ```bash
  npm install
  ``` 

## 🧪 Running the tests

  ```
  npm test
  ```
